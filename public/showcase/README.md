# App screenshots for the marketing site

Add **7** WebP files here for the home **App** section (`#showcase`).

Paths are defined in `lib/showcase-screens.ts`.

## File names (exact)

| # | File |
|---|------|
| 1 | `avatar-dark.webp` |
| 2 | `avatar-light.webp` |
| 3 | `chat-dark.webp` |
| 4 | `chat-light.webp` |
| 5 | `conversation-dark.webp` |
| 6 | `conversation-light.webp` |
| 7 | `shop.webp` |

## Optional 8th file

`shop-light.webp` — set `altSrc: '/showcase/shop-light.webp'` in `SHOWCASE_SHOP` in `lib/showcase-screens.ts` to enable a Dark / Light toggle on the shop card.

## Other formats

If you use PNG or JPEG instead, change the matching paths in `lib/showcase-screens.ts` (e.g. `/showcase/shop.jpeg`).
